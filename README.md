#🤖 Orvyn AI

> Your Intelligent Desktop Companion

Orvyn is an AI-powered desktop assistant built with Python that combines conversational AI with desktop automation, productivity tools, and system utilities. It can answer questions, perform local system tasks, manage files, analyze images, and assist users through a clean graphical interface.

---

## ✨ Features

### 🤖 AI Capabilities
- Conversational AI powered by Gemini
- Context-aware responses
- Coding assistance
- Study support
- Career guidance
- Summarization & explanations

### 🖼️ Vision
- Image analysis
- Scene description
- Visual understanding using Gemini Vision

### 🖥️ Desktop Automation
- Open websites
- Open installed applications
- Search Google
- Search YouTube
- Lock PC
- Take screenshots
- Empty recycle bin

### 📂 File Management
- Create folders
- Create text files
- Find files
- Open located files

### ⚙️ System Utilities
- Battery status
- Weather information
- Calculator
- Internet status
- Volume control
- Shutdown / Restart / Sleep
- CPU / RAM / Disk information

### 🧠 Memory
- Save personal preferences
- Recall stored information

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/Jahnavi-jain20/Orvyn.git

cd Orvyn
```

Create a virtual environment:

```bash
python -m venv venv
```

Activate it:

### Windows

```bash
venv\Scripts\activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## 🔑 Environment Variables

Create a `.env` file:

```env
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
WEATHER_API_KEY=YOUR_WEATHER_API_KEY
```

---

## ▶️ Run

```bash
python main.py
```
